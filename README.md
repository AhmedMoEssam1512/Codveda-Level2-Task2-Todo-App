## 📖 Project Overview

A fully functional To-Do List application built with vanilla JavaScript. This app allows users to add, complete, and delete tasks with data persistence using LocalStorage. Tasks remain saved even after browser refresh, providing a seamless task management experience.

This project fulfills the **Level 2 Task 2** requirements of the Codveda Technologies Web Development Internship.

## ✨ Features

### Core Functionality

- ✅ **Add Tasks:** Type and press Enter or click Add button
- ✅ **Mark as Complete:** Click task text or check button to toggle
- ✅ **Delete Tasks:** Remove tasks permanently with delete button
- ✅ **LocalStorage:** Tasks saved automatically and persist after refresh

### User Experience

- 🎨 **Clean UI:** Modern design with Bootstrap 5 and custom CSS
- 📱 **Responsive:** Works perfectly on mobile and desktop
- ⌨️ **Keyboard Support:** Press Enter to add tasks quickly
- ⚡ **Real-time Validation:** Empty task warning with visual feedback
- 🔄 **Dynamic Updates:** No page reload needed

## 🎥 How to Use

1. **Add a Task:** Type in the input field and press Enter or click "Add"
2. **Mark Complete:** Click the task text or green check button
3. **Delete Task:** Click the red trash button
4. **Refresh Page:** Your tasks will still be there (LocalStorage magic!)
